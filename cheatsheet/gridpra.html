fr Relative unit

In this example, the second column take 60px of the available 100px so the first & third columns split the remaining 40px into the two parts ('1fr' = 50% or 20px)

.grid { display: grid; width: 100px; grid-template-columns: 1fr 60px 1fr; }

The CSS grid relative sizing unit fr is used to split rows and or columns into proportional distance. Each fr unit is a fraction of the grid's overall length and width. If a fixed unit is used along with fr (like pixels for example), then the fr units will only be proportional to the distance left over.

CSS Inline Level Grid

#grid-container { display: inline-grid; }

CSS Grid is a two dimensional CSS layout system. To set an HTML element into a inline-level grid-container use display:inline-grid propert/value. The nested items inside this element are called grid items.

The difference between the values inline-grid and grid is that the inline-grid will make the elements inline while grid will make it block-level element.

CSS grid-auto-rows

The CSS grid-auto-rowsproperty specifies the height of the implicitly added grid rows or it sets a size for rows in a grid container. This property is declared on the grid container. grid-auto-columns provides the same functionality for columns. Implicity-added rows of columns occur when there are more grid items then cell available

CSS unit rem

html { font-size: 18px; } span { font-size: 2rem; }

The CSS unit rem can be used to set the font size of the HTML elements relevant to the font size of the root element. 1 rem represents size of the base font within the root element the html tag.

In the example code block, font size of all the span elements will be twice in size of the font size declared for the root element i.e. html (root element). In other words rem doubles the size of the root element. By default font-size is 16px, so if you mention span elements as 2rem in css. Size represented on user screen will be 32px

CSS flex-wrap property

.container { display: flex; flex-wrap: wrap; width: 200px; }

The flex-wrap property specifies whether flex items should wrap or not. This applies to flex items only. Once you tell your container to flex-wrap, wrapping become priority over shrinking. Flex items only begin to wrap if their combined flex-basis value is greater than the current size of their flex-container.
